Icon Font

A font that is made for being an icon is called an “icon font”. Use the CSS 3 @font-face as the Web Font to call.

Advantage

  • Small file-sizes, improve the performance
  • Easy to Use, full CSS styling capabilities
  • All Browser support (include IE6&7)
  • Automatically scale for retina displays

Disadvantage

  • Not support multi-colors

Real Case

  • twitter
  • github.com
  • BootStrap Framework
  • Font Awesome
  • Tao Bao

Node.js

A JavaScript runtime

http://nodejs.org/

The Details

  • Server-Side JavaScript
  • Google's V8 engine
  • Open Source - MIT License
  • Cross platform

Advantage

  • Single threaded code
  • Non-blocking I/O
  • NPM

Disadvantage

  • Not suitable for compute-intensive applications

What can it do?

  • Build web applications
  • Creating command line tools
  • Create desktop applications (node-webkit)

Who is using it?

PayPal, LinkedIn, Yahoo, Microsoft, eBay, 37signals, LearnBoost, Yammer, Walmart

Responsive Web Designs

随着用户浏览网页的设备越来越复杂(手机,平板,电脑), 之前只针对大尺寸屏幕的web设计,已经不太适合了,我们的设计需在不同设备上,做出不同响应，保证用户良好的体验，这就是响应式设计

响应式设计的真实案例

  • https://ace.ng.bluemix.net/
  • http://colly.com
  • http://foodsense.is
  • http://www.sony.com

你可以通过改变浏览器窗口的大小或chrome dev tools的模拟器，去模拟不同设备尺寸，体验什么是响应式设计。
